# DodoApps Homebrew Tap

This is the official Homebrew tap for DodoApps applications.

## Installation

```bash
brew tap dodoapps/tap
```

## Available Casks

### DodoCount

A beautiful macOS menubar app for Google Analytics 4 and Search Console.

```bash
brew install --cask dodoapps/tap/dodocount
```

Or after tapping:

```bash
brew install --cask dodocount
```
